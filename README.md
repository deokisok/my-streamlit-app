ootd는 “일일 코디 결정 비용”을 줄이는 데 초점을 둔 AI 추천 앱이다.
사용자는 자신의 옷을 사진으로 등록하고, 앱은 날씨·상황·사용자 취향 데이터를 통합해 코디를 추천한다. 

주요기능)
1. 옷장 등록: 사진 업로드
카테고리 선택(top/bottom/outer/shoes)
(옵션) AI Vision 분석 ON → color/pattern/warmth/vibe 저장
(옵션) 스타일 태그(주/보조) 선택 가능
영수증으로 업로드: problem)영수증에서 항목 추출은 가능하지만 이미지 생성에 차질
2. 날씨 자동 연동: Nominatim으로 좌표 → 주소 표시
3. 코디 추천 엔진: 후보 조합 생성(top/bottom/shoes [+ outer])
점수 구성:
날씨 점수(두께, 아우터 필요성)
상황 적합 점수(격식/운동/데이트 등)
색 조합 규칙 점수(뉴트럴 안정성, 멀티 과다 감점)
패턴 조합 규칙 점수(패턴 1개 포인트, 패턴 과다 감점)
분위기(vibe) 적합 점수(상황 기반)
 사용자 취향 학습 점수(선호/비선호 누적 기반)
4. 피드백 수집/학습
피드백 수집/학습
 별점(1~5): 추천 품질의 총합 평가
 온도 체감: 추움/딱좋음/더움 → 온도 보정값 학습(temp_bias)
 색/ 패턴/ vibe: 좋음/별로/상관없음 → 선호·비선호 프로필 누적
결과: 다음 추천에서 선호 요소는 가산, 비선호는 감점
5. 로그인 기능
아이디를 저장해 옷장 저장 가능

기술 스택)
Streamlit: UI/상태/페이지 구성
OpenAI API)
Vision: 사진 → 메타 추출
LLM: 후보 리랭크 + 이유 생성
Open-Meteo(무료): 날씨 자동 연동
OSM Nominatim(무료): 위치 문자열 표시
Pillow(PIL): 이미지 저장/기본 이미지 생성
JSON 파일 기반 저장: 사용자별 로컬 DB 역할
